Scaled Dot-Product Attention

30秒看懂

Attention 是一个按相关性加权取信息的机制。

5分钟讲懂

让每个 token 根据相关性从其他 token 聚合信息。

公式

$$ \mathrm{Attention}(Q,K,V)=\mathrm{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V $$

输入

  • Q: 当前 token 想查询什么
  • K: 每个 token 提供什么匹配线索
  • V: 每个 token 提供什么内容

输出

  • 按相关性加权后的上下文表示

每一项在做什么

符号含义作用去掉会怎样权重变化
QK^Tquery 与 key 的匹配分数鼓励相关 token 互相连接无法判断该看谁
\sqrt{d_k}维度缩放避免点积过大导致 softmax 过尖缩放变小会让分布更尖
V被聚合的内容把注意力权重转成实际上下文表示
深挖推导

这个公式把“找相关 token”和“聚合信息”合成一个可微模块。

若两个 key 得分分别为 2 和 0，softmax 后第一个 token 权重更高。
