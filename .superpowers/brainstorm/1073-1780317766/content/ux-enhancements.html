1. 阅读舒适度优化

Font / Theme / Spacing / Focus Mode
字体 & 间距
  • 正文字号 17px，行高 1.75，段间距 16px
  • 标题用思源宋体/霞鹜文楷，正文用 Noto Sans SC
  • 公式用 KaTeX 渲染，字号比正文大 2px
  • 中英混排自动调整间距
  • 最大内容宽度 720px（最佳阅读宽度）
🌙 暗色模式
  • 背景 #0f172a（非纯黑），文字 #e2e8f0
  • 卡片背景 #1e293b，边框 #334155
  • 公式区背景 #1a2332
  • 跟随系统 or 手动切换
  • 记住用户偏好
🎯 专注模式： 按 F 键进入，隐藏左右栏，只保留中间内容区 + 底部迷你追问栏。再按 Esc 退出。适合深度阅读时减少干扰。

3. 智能导航 & 定位

Smart Nav + Breadcrumb + Search + Back/Forward
🏠 首页 › 方向：RAG可信性 › RAG 基石论文 › 公式：检索增强损失
功能点：
  • 面包屑导航：方向 → 论文 → 卡片 → 公式，点击可跳回
  • 前进/后退：记住阅读路径，像浏览器一样

4. 书签 & 高亮 & 笔记

Highlight + Bookmark + Notes + Export
注意力机制的核心假设是 每个 token 与所有其他 token 都有关联， 但实际中大多数关联很弱。 这就是稀疏注意力的出发点。
📌 黄色=重要 📌 蓝色=概念 📌 绿色=创新点 📌 粉色=疑问
📌 书签
标记重要段落/公式，侧边栏快速跳转
支持分类：重要/疑问/复习
📝 批注
选中文字添加自己的理解笔记
笔记与原文绑定，支持导出
