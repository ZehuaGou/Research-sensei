# 09_GLOSSARY

## A_READ
必须精读的论文，会进入全文获取与深度解析。

## B_SKIM
略读论文，只生成简要卡或保留摘要。

## Block
论文解析后的最小证据单元。可以是段落、公式、表格、图、算法、参考文献等。

## Evidence Ref
证据引用 ID，例如 `paper_xxx:b042`，用于把解释绑定到原文。

## Grounding
把系统生成的 claim 和原文证据绑定的过程。

## Paper Skeleton
论文骨架理解 JSON，包括问题、旧方法、瓶颈、新假设、机制、公式、实验、局限、迁移点。

## Formula Card
公式讲解卡，解释公式目的、符号、每一项作用、去掉会怎样、数字例子。

## Pattern Card
科研模式卡，判断论文创新属于表示、目标函数、结构、评估、系统管线等哪类。

## Drill Card
训练卡，包括复述题、复习题、迁移题、导师追问。

## Context Pack
用户追问时构造的上下文包，包含当前卡片、选中文本、证据、历史对话、用户画像和隔离后的问题。

## Advisor Mode
模拟导师追问的多轮状态机，不是一次性问答。

## Degradation
降级输出。当材料不足时，只能输出低置信度或基础摘要，不能假装深度理解。
